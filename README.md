# 🌱 Eco Cultivate

Eco Cultivate is a web-based platform designed to connect **farmers, customers, and landlords** in a unified system.
It helps farmers sell products, customers purchase fresh produce, and landlords provide land for cultivation.

---

## 🚀 Features

### 👨‍🌾 Farmer

* Upload and manage products
* View customer orders
* Manage product listings

### 🛒 Customer

* Browse available products
* Place orders easily
* View product details

### 🏡 Landlord

* Upload available land
* View farmer interest requests
* Manage land listings

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP
* **Database:** MySQL
* **Tools:** XAMPP, VS Code

---

## 📁 Project Structure

* `index.html` – Homepage
* `farmer.html` – Farmer dashboard
* `customer.html` – Customer interface
* `landlord.html` – Landlord interface
* `backend/` – PHP files
* `database/` – SQL files

---

## 🎯 Objective

The main goal of Eco Cultivate is to:

* Support farmers by giving them a direct selling platform
* Provide customers with fresh and local produce
* Utilize unused land efficiently through landlord participation
